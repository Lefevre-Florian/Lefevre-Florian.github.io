Landsliders

Incarnez le leader d'une colonnie de nomade voyage à travers le monde, progresser en réussissant vos quêtes, dans ce jeu de cartes, où vous altérer directement l'environement autour de vous.

Catégorie

Free to play / Jeu de cartes / Jeu de réflexion

Date de fin de projet

06/2024

Informations de développement

Unity 2D / C#

Support

Mobile (Android)

Programmation

  • Programmation gameplay (Système de biomes)
  • Programmation de la FTUE (Tutoriel) + Système de dialogue intégré
  • Programmation de l'UI
  • Programmation d'un système de sauvegarde et de chargement dynamique d'asset via SQL et Unity Addressable
  • Intégration des events de tracking via Unity Data Analytics
  • Intégration + Programmation des triggers sons via FMOD
  • Debug et optimisation
